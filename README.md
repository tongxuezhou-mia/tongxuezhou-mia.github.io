# Tongxue Zhou Research Homepage

This repository is a static GitHub Pages homepage for Tongxue Zhou.

## Publish With GitHub Pages

1. In the GitHub account `tongxuezhou-mia`, create a repository named:
   `tongxuezhou-mia.github.io`

2. Upload these files and folders to the repository root:
   - `index.html`
   - `styles.css`
   - `assets/profile.jpg`
   - `.nojekyll`

3. In GitHub, open the repository and go to:
   `Settings` -> `Pages`

4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Your site will be available at:
   `https://tongxuezhou-mia.github.io/`

## Updating Content

- Edit `index.html` to update biography, publications, projects, teaching, and awards.
- Edit `styles.css` to update the visual design.
- Replace `assets/profile.jpg` to update the profile photo while keeping the same filename.
